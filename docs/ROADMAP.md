## Removed (2026-07-10): Team/Assignee Features

WorkManager is a single-user personal tool; the team roster, per-member capacity, workload/reassignment UI, and mandatory task ownership built up over earlier iterations didn't fit that scope and were removed at the user's request (see docs/IMPROVEMENT_PLAN.md Wave 2 item 21).
